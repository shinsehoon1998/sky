# 고객정보 등록 및 동의서 출력 업무 자동화 시스템

## 프로젝트 개요

보험 영업용 전산 업무 자동화 시스템입니다. 엑셀 파일에 저장된 고객 정보(주민등록번호, 이름, 연락처)를 업로드하면 자동으로 데이터를 검증하고, 클립보드 매크로 방식으로 보험사 전산 시스템에 입력을 지원하며, 고객별 개인정보 수집·이용 동의서를 생성·출력합니다.

### 주요 기능

- **엑셀 파일 업로드 및 파싱**: .xlsx 파일을 드래그 앤 드롭으로 업로드, A/B/C열 데이터 자동 추출
- **데이터 검증**: 주민등록번호(13자리/생년월일/성별코드), 이름(특수문자), 연락처 형식 자동 검증
- **데이터 미리보기**: 페이지네이션, 검색/필터링, 체크박스 선택, 오류 하이라이트
- **클립보드 매크로**: 고객 데이터를 클립보드에 순차 복사하여 보험사 전산 시스템에 붙여넣기
- **동의서 생성**: 고객별 개인정보 수집·이용 동의서 자동 생성 (HTML 템플릿)
- **동의서 출력/PDF**: 현재 동의서 인쇄, PDF 다운로드, 선택 고객 일괄 인쇄

## 사용 방법

### 1. 엑셀 파일 업로드
- 메인 화면의 드롭존에 .xlsx 파일을 드래그하거나 클릭하여 선택
- A열(주민등록번호), B열(이름), C열(전화번호) 데이터가 자동 추출됨

### 2. 데이터 확인 및 검증
- 자동 검증 결과 확인 (정상/오류 건수)
- 오류가 있는 데이터는 빨간색 하이라이트로 표시
- 검색 및 필터링으로 원하는 데이터 찾기
- 처리할 고객 선택 (체크박스)

### 3. 전산 등록 매크로
- **클립보드 방식**: "다음 고객 복사" 버튼 → 보험사 전산 시스템에서 Ctrl+V로 붙여넣기
  - 단축키: `Ctrl+→` (다음 고객), `Ctrl+←` (이전 고객)
- **전체 복사**: 모든 선택 고객 데이터를 탭 구분으로 한 번에 복사

### 4. 동의서 출력
- 고객별 동의서 미리보기
- "현재 동의서 인쇄" 버튼으로 브라우저 인쇄
- "현재 동의서 PDF" 버튼으로 PDF 다운로드
- "선택 고객 일괄 인쇄"로 모든 선택 고객 동의서 한 번에 인쇄

## GDPR 컴플라이언스

본 시스템은 GDPR(EU 일반개인정보보호법) 및 대한민국 개인정보보호법을 준수하도록 설계되었습니다.

- **데이터 최소화**: 필수 3개 컬럼만 처리, 불필요 컬럼 무시
- **암호화**: HTTPS 통신, 주민등록번호 마스킹 처리
- **보관 기간 제한**: 처리 완료 후 서버 데이터 즉시 삭제
- **Privacy by Design**: 설계 단계부터 개인정보보호 원칙 적용
- **동의 기반 처리**: 동의서를 통한 명시적 동의 획득
- **데이터 주체 권리**: 열람·정정·삭제 권리 지원
- **상세 내용**: PRD.md Section 15~17 참조

## 데이터 모델

| 컬럼 | 필드명 | 데이터 타입 | 비고 |
|------|--------|------------|------|
| A | 주민등록번호 | String(13) | 필수, 마스킹 표시 |

| C | 전화번호 | String(13) | 필수, 010-XXXX-XXXX 표준화 |

## 제한사항

- Cloudflare Workers 환경으로 인해 서버 측 브라우저 자동화(Playwright) 불가
- 보험사 전산 시스템 직접 API 연동이 아닌 클립보드 매크로 방식 사용
- iframe을 통한 전산 시스템 접속은 대상 사이트의 X-Frame-Options 정책에 따라 차단될 수 있음
